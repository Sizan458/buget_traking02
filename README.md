# Expense Tracker App

A simple and modern expense tracking app to help you manage your budget, track spending, and stay in control of your money.

This project focuses on clarity and usability. You can create budgets, log expenses, and instantly see where your money is going through a clean dashboard with charts and summaries.

## Features

- Dashboard overview with total budget, total spend, and number of budgets  
- Create and manage multiple budgets  
- Track expenses under each budget  
- Visual charts to understand spending patterns  
- Remaining budget calculation in real time  
- Clean and responsive UI  
- User-friendly layout with sidebar navigation

## Tech Stack

- **Frontend:** React / Next.js  
- **Styling:** Tailwind CSS  
- **Charts:** Chart library (bar chart for budget activity)  
- **Backend:** Node.js  
- **Database:** Drizzle ORM
